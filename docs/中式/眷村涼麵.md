## 👨‍🍳 步驟

### 1. 蒜汁醬
1. 大蒜搗成泥
2. 加約半杯冷開水
3. 加入醬油、胡椒、糖、味精、醋、麻油
4. 攪拌均勻

### 2. 蛋皮
1. 2 顆蛋加鹽、料酒、胡椒、太白粉水
2. 打散（不要打發）
3. 不沾鍋噴少許油，**中小火**
4. 倒入蛋液，轉一轉讓蛋液均勻
5. 約 **10-20 秒**翻面
6. 煎到兩面黃黃的有**焦香**才香
7. 切絲備用

### 3. 芝麻醬（傳統，不放花生醬）
1. 2 大匙芝麻醬先用瓶裡的油（或麻油）打散
2. **一定要用冷開水慢慢加**，一次半湯匙，邊加邊攪
3. 加了水會先變稠，繼續加到流質狀態（約加 4 次）
4. 最後加 1 大匙麻油、1/2 茶匙醬油調色

### 4. 花椒蔥油
1. 蔥洗淨晾乾（或沖水瀝乾防炸鍋）
2. 薑切薄片 4-5 片
3. 一杯油**中小火**，先放薑片，再放蔥
4. 炸到蔥變黃後放花椒 **3-5 秒**即撈出
5. 💡 這個油可以拌涼菜、當明油，家裡常備

### 5. 煮麵
1. **水要多**，煮出來才清爽
2. 煮到**八九分熟**，中間還有一點白芯即可
3. 撈出瀝乾，倒**蔥油拌勻**防黏
4. 用**電風扇吹涼**，不要過冷水（過冷水不好吃）

### 6. 組合
1. 先放**醬汁**
2. 再放**蛋皮絲、小黃瓜絲、榨菜末、蔥花**
3. 喜歡辣可加**辣油**
4. 醋不夠可以再加，更清爽

## 🎯 主廚秘訣

- 🔑 **不放花生醬** — 傳統眷村版純用芝麻醬
- 🔑 **甜酒釀 + 花椒油** — 關鍵靈魂，濃郁層次感
- 🔑 **蒜用搗的** — 不用剁或攪碎機，香氣不同
- 🔑 **蛋皮煎到焦香** — 兩面黃黃的才有香氣
- 🔑 **芝麻醬慢慢加水** — 一次半湯匙，邊加邊攪
- 🔑 **花椒炸 3-5 秒即撈** — 久了會苦
- 🔑 **麵不要過冷水** — 電風扇吹涼才好吃
- 🔑 **水要多** — 煮出來才清爽不黏
