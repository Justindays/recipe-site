## 📝 配方（可做 3 條）

| 材料 | 用量 | 備註 |
|------|------|------|
| 🌾 高筋麵粉 T65 | **500 g** | 無添加物 |
| 🧂 鹽 | **9 g** | |
| 🧫 新鮮酵母 | **4 g** | 或乾酵母 **2 g** |
| 💧 水 | **330 g** | 溫度需計算（見下方） |
| 🫘 天然酵種 | **50 g** | 選擇性，主廚用蘋果+蜂蜜自養6年 |

## ⚙️ 關鍵技巧

### 1️⃣ 水溫計算 — 基礎溫度法

> **目標麵團溫度：23~26°C**
> 公式：**基礎溫度 − 室溫 − 麵粉溫度 = 水溫**

**範例：**
```
基礎溫度 65°C
− 室溫 26°C
− 麵粉溫度 27°C
= 水溫 12°C ❄️
```

### 2️⃣ 自解法（Autolyse）— ⏱ 45分鐘

- 水 + 麵粉，鉤狀頭慢速攪 **3 分鐘**
- 保鮮膜蓋住，**室溫靜置 45 分鐘**
- 目的：麵粉太強時改善延展性

### 3️⃣ 攪拌

- 加入鹽、酵母、酵種
- **低速8分鐘 + 中速2分鐘**
- 太乾可加少量水
- ⚡ **窗玻璃測試（Window pane test）**：確認麵筋網絡形成

### 4️⃣ 基礎發酵（Pointage）— ⏱ 1小時室溫

- 放入容器，做一次**折疊（stretch & fold）**
- 然後放入冰箱 **4°C，冷藏12小時** ❄️
- 目的：增加風味與保存性

### 5️⃣ 隔日步驟

1. 取出麵團回溫至 **17°C**
2. 分割 **每份 300 g**
3. 預整形成 batard（橢圓形）
4. 撒薄粉，**靜置 15 分鐘**（蓋住）
5. **整形**成棍形，接縫朝上放發酵布（couche）上
6. 撒粉，輕拍排掉發酵氣泡
7. 接縫朝下放回發酵布

### 6️⃣ 最後發酵（Second proof）— ⏱ 45 分鐘室溫

- 記得蓋住麵團

### 7️⃣ 割線（Scoring）

- 預熱烤箱 **250°C**（可放披薩石板）
- 用**刮鬍刀片**（非廚房刀）
- **30公分的長棍割3刀**
- 撒上白麵粉裝飾

### 8️⃣ 烤焙

| 步驟 | 時間 | 溫度 |
|------|------|------|
| 🔥 帶蒸氣烤 | **15 分鐘** | 250°C |
| 🌬️ 釋放蒸氣，續烤 | **3~4 分鐘** | 250°C |

**自製造蒸氣：**
1. 燒一壺熱水
2. 打開烤箱門
3. 麵團放進石板
4. 底部烤盤倒入熱水 → **快速關門**

> 💡 蒸氣是長棍漂亮色澤的關鍵！

### 9️⃣ 冷卻

- 置於網架**徹底放涼**

## 🎯 主廚秘訣總整理

| 秘訣 | 說明 |
|------|------|
| 🌡️ **控溫** | 水溫/室溫/麵粉溫度都影響發酵 |
| 🧪 **窗玻璃測試** | 確認麵筋是否到位 |
| 🧊 **低溫長時間發酵** | 12小時冷藏，風味更佳 |
| 💨 **蒸氣** | 麵包上色與外皮酥脆的關鍵 |
| 🧑‍🍳 **工作檯保持乾淨** | 隨時用刮板清潔，養成好習慣 |

## 🥖 成功長棍的樣貌

- 外皮：⚡ **酥脆聲**（crunchy sound）
- 內部：🍯 **蜂巢狀氣孔**（honey comb structure）
- 香氣：天然發酵的麥香
