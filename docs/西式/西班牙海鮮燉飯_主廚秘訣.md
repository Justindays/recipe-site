## 這支影片的特色

> 西班牙主廚親自教學，不添加番紅花也能做出美味海鮮燉飯！

---

## 📝 食譜重點

### 1️⃣ Sofrito（醬底）— 風味的基礎

| 材料 | 說明 |
|------|------|
| 洋蔥碎 | 先爆香 |
| 紅椒 | 切碎加入 |
| 青椒 | 切碎加入 |

> 可**前一天先做好**，隔天再完成燉飯

### 2️⃣ 高湯（Caldo）— 最關鍵的一步

- 用**蝦殼、海鮮殼**熬煮高湯
- 高湯是燉飯好吃的靈魂
- 建議自熬，不要用水代替

### 3️⃣ 米與高湯比例

| 材料 | 比例 |
|------|------|
| 🔸 米 | **4 份** |
| 🔹 高湯 | **8 份** 標準比例 |
| 🔹 實際用量 | **7 份**（因 sofrito 本身含水分） |

### 4️⃣ 祕密食材

- 🍅 **番茄** — 增添色澤與風味，代替番紅花
- 🍷 **白酒** — 用料理用白酒烹調
- 🧂 **不加鹽** — 海鮮與醬底本身風味已夠濃郁

### 5️⃣ 烹調步驟

1. 先做 **Sofrito**（洋蔥+紅椒+青椒炒軟）
2. 加入**番茄**增加色澤
3. 倒入**白酒**拌炒
4. 加入**高湯**（米與高湯約 4:7）
5. 放入**海鮮**（蝦、蛤蜊等，先處理好去殼）
6. 爐火先煮一陣
7. 進 **200°C 烤箱**烤至完成

### 6️⃣ 主廚小撇步

- 🦐 **海鮮先去殼** — 讓客人吃的時候更方便，不用動手剝
- 📅 **Sofrito 可提前備** — 分天做更輕鬆
- 🌡️ **先爐火後烤箱** — 米粒口感更佳
- 🚫 **不塗番紅花**也能做出漂亮色澤
