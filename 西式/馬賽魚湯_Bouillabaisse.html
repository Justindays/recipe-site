馬賽魚湯 Bouillabaisse

法餐經典：馬賽魚湯 Bouillabaisse — 天鵝絨般絲滑版 ​

這支影片的特色 ​

打破馬賽魚湯又腥又鹹的刻板印象！結合 Velouté 技巧，將蔬菜打碎細濾，打造絲滑如天鵝絨般的極致湯頭。

步驟 ​

Part 1：海鮮處理 ​

龍蝦分解（徒手法） ​

  1. 用毛巾抓住龍蝦，一手握頭、一手握尾 → 一扭分離
  2. 鉗子同樣扭下
  3. 龍蝦尾：用毛巾包住，手掌壓破殼 → 剪刀剪開取肉
  4. 龍蝦頭：從中間掰開，取出 蝦腦（極鮮，留用）
  5. 去除龍蝦鰓
  6. 鉗子：用杯子/鈍器敲碎（毛巾包住避免飛濺）
  7. 所有殼留用 → 熬海鮮高湯

紅鯛魚處理 ​

  1. 剪掉魚鰭（避免刺手）
  2. 廚房紙擦乾表面
  3. 沿魚頭切開 → 沿脊骨片下魚肉
  4. 用鑷子拔除魚刺
  5. 魚頭+魚骨留用 → 熬高湯
  6. 💡 魚鱗不刮 — 煎的時候會形成酥脆外殼

魷魚處理 ​

  1. 輕輕劃開表面薄膜 → 水下撕掉
  2. 分離尾巴與身體
  3. 取出透明骨骼（海螵蛸）、墨囊、眼睛、嘴巴
  4. 切約 1 公分寬的魷魚圈

貝類處理 ​

  1. 鹽水浸泡，刷洗表面
  2. 去除海草/鬚
  3. 已開口不新鮮的 → 丟棄

Part 2：海鮮高湯（魚骨+龍蝦殼） ​

  1. 橄欖油熱鍋 → 放入 魚骨+龍蝦殼 翻炒至變色
  2. 加入少許蔬菜（西餐三寶各一把）
  3. 加白葡萄酒 Deglaze（刮起鍋底精華）
  4. 加水 完全淹沒食材
  5. 加入 香草束（百里香+月桂葉+歐芹莖+龍蒿）
  6. 大火煮開 → 轉小火，打掉浮沫
  7. 加幾粒黑胡椒
  8. 小火微沸 40 分鐘（不蓋蓋子）
  9. 過濾 → 只取清湯

Part 3：Rouille 紅椒醬 ​

  1. 紅椒放火上燒烤至表皮焦黑
  2. 放入碗中，蓋保鮮膜燜 5-10 分鐘
  3. 刮掉燒焦表皮，去籽去筋膜
  4. 紅椒切粒 + 1 瓣大蒜 + 鹽 + 紅椒粉 + 半顆檸檬汁 + 1 顆蛋黃
  5. 手持攪拌棒打至順滑
  6. 緩慢加入橄欖油（像做美乃滋，乳化）
  7. 也可手打（打到打不動為止）

Part 4：調味底湯 ​

  1. 黃油融化 → 炒蔬菜丁（洋蔥、西芹、紅蘿蔔、茴香、韭蔥）
  2. 加一撮鹽，炒到洋蔥透明
  3. 加入 蒜末、蝦腦、番茄膏
  4. 白葡萄酒 Deglaze → 刮起鍋巴
  5. 加入 橙皮、海鮮高湯、番紅花、百里香
  6. 大火煮開 → 轉小火 咕嘟 20 分鐘
  7. 取出百里香
  8. 手持攪拌棒打碎所有蔬菜 → 至無顆粒（天鵝絨質地）

Part 5：組合與擺盤 ​

  1. 底湯重新燒開
  2. 放入 龍蝦頭+鉗子+所有貝類 → 大火煮
  3. 貽貝開口後立即撈出（不能煮老）
  4. 放入 龍蝦肉 → 快速汆燙
  5. 過濾湯汁（可選，餐廳做法求細膩）
  6. 魚塊撒鹽 → 油鍋煎至表皮金黃酥脆（不要馬上翻面）
  7. 將所有海鮮放回湯中
  8. 法棍切片 → 抹黃油 → 200°C 烤 5 分鐘

擺盤 ​

  1. 深盤盛湯
  2. 擺上煎魚、龍蝦頭、貽貝、蛤蜊、魷魚圈
  3. 插兩片烤法棍
  4. Rouille 紅椒醬放旁邊
  5. 點綴小歐芹

主廚秘訣 ​

  • 🔑 魚鱗不刮 — 煎出酥脆外殼，口感升級
  • 🔑 蝦腦留用 — 極鮮，放入湯中風味加倍
  • 🔑 蔬菜打碎 — 天鵝絨般絲滑湯頭的關鍵
  • 🔑 番紅花 — 香氣與漂亮紅色的來源（貴但值得）
  • 🔑 Rouille 紅椒醬 — 沾麵包或混入湯中都絕配
  • 🔑 法棍吸湯 — 吸滿魚湯精華，超級有靈魂
